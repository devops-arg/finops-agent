FinOps Agent · DevOps ARG

                
AWS cost optimization specialist · read-only, multi-round reasoning

              

            

            

              
Hi 👋 — I'm the FinOps Agent from DevOps ARG.

              
My job is to analyze your AWS account and find where you're overspending. I have read-only access to Cost Explorer, infrastructure APIs (EC2/RDS/EKS/ElastiCache/S3/Lambda), Compute Optimizer, Cost Optimization Hub, and the Savings Plans/RI recommender. I can also run any describe-* / list-* / get-* AWS command live via call_aws.

              
I won't invent numbers. If I quote a cost, it came from a real Cost Explorer call and I can show you the exact query I ran.

              
Try one of the preset questions in the sidebar ←, or just ask freely: "What were my top 3 services last week?", "Any wasted NAT Gateways?", "Which old snapshots can I safely delete?"

              
English by default · also fluent in Spanish, just write to me that way.
